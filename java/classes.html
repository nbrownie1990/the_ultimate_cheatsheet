Arten von Variablen lokal Beispiel statisch Jedes Objekt hat eine eigene Kopie der Instanzvariablen, alle Objekte teilen sich jedoch nur eine Kopie der statischen Variablen Methoden Methoden bestehen aus Modifikator z.B. private Öffentlichkeit Rückgabetyp Methodenname Parameter Methodenkörper / Logik statische Methoden gehören zu einer Klasse, es muss kein Objekt instanziiert werden, um die Methode zu verwenden Methodensignatur Die Methodensignatur enthält NICHT()) : Zugriffsmodifikator und Rückgabetyp In Java können Sie nicht >= 2 Methoden mit derselben Methodensignatur in derselben Klasse haben (selbst wenn Methoden unterschiedliche Rückgabetypen haben).

Abstrakte Klasse Objekt der abstrakten Klasse kann nicht erstellt/instanziiert werden Jede Unterklasse der abstrakten Klasse muss entweder alle abstrakten Methoden implementieren oder selbst als abstrakt deklariert werden. Aufgrund der teilweisen Implementierung von Klassen kann das Objekt nicht instanziiert werden. Die abstrakte Klasse hängt von der Unterklasse ab, um eine vollständige Implementierung bereitzustellen.
