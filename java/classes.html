Datentypen

Überschrift
In Programmiersprachen haben wir verschiedene Arten von Daten mit denen gearbeitet wird. Primitive Datentypen Primitive Datentypen in Java werden von der Java-Sprache vordefiniert und als reservierte Schlüsselwörter benannt. Ein primitiver Datentyp teilt keinen Zustand mit anderen primitiven Werten. Schauen wir uns alle 8 primitiven Datentypen an, die von Java unterstützt werden.

  • byte: (byte x=10;)
  • short: (short x=10;)
  • int : Wird zum Speichern von Ganzzahlen verwendet.(int x=10;)
  • long: Wird verwendet, um große ganzzahlige Werte zu speichern (long x=10000000;)
  • float: Wird zum Speichern von Dezimalzahlen verwendet (float x=10f;)
  • double: Wird verwendet, um große Dezimalzahlen zu speichern (double x=10.1010;)
  • boolean: Die Datentypen, die entweder true oder false Werte haben. (boolean x=1;) Dies ist sehr nützlich beim Schreiben von bedingten Anweisungen.
  • char: Wird verwendet, um ein einzelnes Zeichen oder einen Buchstaben zu speichern (char c ='s';)
Nicht-primitive Datentypen Nicht primitive Datentypen werden Referenztypen genannt, weil sie auf Objekte verweisen. Es gibt viele nicht primitive Datentypen in Java: Arrays, Strings , Klassen usw. Sehen wir uns an, wie man ein Array und einen String initialisiert.
  • Array: Eine lineare Datenstruktur die Werte an zusammenhängenden Speicherorten speichert. (int[] array = new int[10];)
  • String: (String s = "hello world";)
literal: Dies sind die syntaktischen Darstellungen unserer Daten. In sehr einfacher Sprache sind dies die tatsächlichen Werte unserer Variablen identifier: Name der Variablen, Klassen, Methoden, Funktion sind alle als Bezeichner bekannt. Wenn wir also eine Variable deklarieren, ist ein Variablenname ein Bezeichner Typkonvertierung & Casting Wenn Sie versuchen, eine Dezimalzahl oder einen Gleitkommawert in eine String-Variable einzugeben, erhalten Sie eine Fehlermeldung und das Programm schlägt fehl. Dasselbe gilt, wenn Sie eine Gleitkommazahl in eine Variable vom Typ int eingeben. Wenn Sie andererseits eine Dezimalzahl für Float-Daten angeben, wird die Dezimalzahl automatisch in Float konvertiert.

Bedingungen für die Typkonvertierung Zwei Typen sollten kompatibel sein char & String int & float Der Zieltyp sollte größer als der Quelltyp sein

float ist größer als intweil sie neben ganzen Zahlen auch Dezimalzahlen enthält. Statisch typisiert Führen Sie zur Kompilierungszeit eine Typprüfung durch bietet Typensicherheit Wenn Java-Code Fehler enthält, kann er nicht kompiliert werden, bis die Fehler behoben wurden deklarieren Sie Datentypen von Variablen, bevor Sie sie verwenden int num; num = 5; VS num =5 erwartet, dass Variablen deklariert werden, bevor ihr ein Wert zugewiesen werden kann. Das Gegenteil von statisch typisiert ist dynamisch typisiert (z. B. Python)

Abstrakte Klasse Objekt der abstrakten Klasse kann nicht erstellt/instanziiert werden Jede Unterklasse der abstrakten Klasse muss entweder alle abstrakten Methoden implementieren oder selbst als abstrakt deklariert werden.
