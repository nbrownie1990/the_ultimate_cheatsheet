Was ist Destructuring?

Die destrukturierende Zuweisungssyntax ist ein JavaScript-Ausdruck, der es ermöglicht, Werte aus Arrays oder Eigenschaften aus Objekten in unterschiedliche Variablen zu destrukturieren/zu entpacken.

1. Destructing

Das Destructing ("Zerstören") eines Objekts oder Arrays bedeutet nicht, dass wir es aus dem Programm löschen und nie wieder verwenden können, sondern dass wir einen bestimmten Teil davon erhalten. Destrukturieren extrahiert einfach Array-Elemente oder Objekteigenschaften und speichert sie in Variablen. Beim Destrukturieren des Arrays wird der Wert des Index gemäß der Indexierung mit dem linken Key zurückgegeben.

2.The Spread Operator(...)

Der Spread-Operator wird verwendet, um Array-Elemente oder Objekteigenschaften aufzuteilen. Es führt eine deep copy der Array-Elemente oder des Objekts durch, sodass die ursprünglichen Werte nicht beeinflusst werden. Achtung: Die deep copy gilt nur für eine Leveltiefe - z.B. nicht bei Objekte innerhalb eines Objekts. Denn dort werden die nested Objects wieder als Referenz-Typ übertragen. Um eine vollständige deep Copy (inkl. nested Objects) durchzuführen, muss JSON.parse/stringify. "JSON.parse(JSON.stringify(object))" durchgeführt werden. Oder man kann auch andere Libraries nehmen.

Nehmen wir das Array als Flasche. Der Spread-Operator ermöglicht es Ihnen, den gesamten Inhalt der Flasche zu greifen, ohne die Flasche selbst zu greifen und diesen Inhalt an einer beliebigen Stelle zu platzieren. Der Spread-Operator kann verwendet werden, um den Inhalt eines Arrays oder mehrerer Arrays in ein anderes Array zu kopieren.

    Beispiel
  • let arr= [1,2,3]
  • let arr1= [...arr,4,5,6]
  • console.log(arr1) /// [1,2,3,4,5,6]

3. The Rest Operator (...)

Der Rest Operator ist das Gegenteil vom Spread-Operator. Denn im Gegensatz zum Spread-Operator, der die Elemente in einem Array verteilt, ist es beim Rest-Operator(auch Gather-Operator genannt) so, dass er mehrere Elemente in einem Array gruppiert/sammelt. Der Rest-Operator wird in einer Funktion immer dann verwendet, wenn mehrere Arguments erwartet werden.

    Beispiel
  • function multiply(multiplier, ...theArgs){
  • return theArgs.map(function(element){
  • return multiplier * element;
  • });
  • }

  • var arr = multiply (2,1,2,3)
  • console.log(arr)
  • Result: [2,4,6]
