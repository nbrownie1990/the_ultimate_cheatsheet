The Hash Table

Was ist eine Hash Table?

Hash Tables werden in verschiedenen Programmiersprachen unterschiedlich bezeichnet, sind aber ein und die selbe Sache. In Java werden Hash Tables "HashMap" genannt, in JavaScript "Objects" und in Python oder C# "Dictionarys". Die HashTable ordnet einem Schlüssel Werte zu. Jeder Schlüssel ist einzigartig und kann nur einem Wert zugeordnet werden. Sie ist deshalb super schnell in LookUps und wir können sie nutzen, um unsere Algorithmen zu optimieren.

Working with Arrays

Es gibt 4 Map-Typen in Java, die wie folgt beschrieben werden:

HashMap - Es handelt sich um eine ungeordnete und unsortierte Karte und ist daher eine gute Wahl, wenn die Reihenfolge nicht betont wird. Eine HashMap lässt einen NULL-Schlüssel und mehrere NULL-Werte zu und behält keine Einfügereihenfolge bei.

HashTable – Erlaubt nichts Null und hat Methoden, die synchronisiert werden. Da es die Thread-Sicherheit ermöglicht, ist die Leistung langsam.

LinkedHashMap – Langsamer als eine HashMap, behält aber die Einfügereihenfolge bei und hat eine schnellere Iteration.

TreeMap– Eine sortierte Map, die das Erstellen einer Sortierreihenfolge mithilfe eines Konstruktors unterstützt.
